A professional, customizable notice bar system for WordPress with countdown timer support. Display important announcements, promotions, or notifications at the top of your website.

== Description ==

WP Notice Bar is a powerful and flexible notice bar plugin that allows you to display important messages, announcements, or promotions at the top of your WordPress website. With its smooth animations, countdown timer feature, and responsive design, it's perfect for engaging your visitors with timely information and creating urgency for your promotions.

= Key Features =

* **Easy Setup**: Configure your notice bar in minutes with our intuitive admin interface
* **Visual Color Customization**: Built-in color pickers for easy styling
* **Real-time Color Preview**: See your color changes instantly
* **Responsive Layout**: Looks great on all devices and screen sizes
* **Smooth Animations**: Professional entrance and exit animations
* **Position Control**: Display at the top or bottom of your site
* **Display Rules**: Control where and when your notice bar appears
* **Multiple Notices**: Create and manage multiple notice bars
* **Mobile Friendly**: Optimized for mobile devices
* **Countdown Timer**: Display dynamic countdown to create urgency
* **Scheduling System**: Set start and end dates for your notices
* **Background Options**: Choose between solid colors, gradients, or images
* **Custom Opacity**: Control background transparency
* **Dynamic Spacing**: Automatically adjusts to your theme's header

= Countdown Timer Features =

* Shows remaining days, hours, minutes, and seconds
* Automatically hides notice when countdown ends
* Clean, modern design that matches your theme
* Optional display - enable only when needed
* Works with scheduling system
* Perfect for:
  * Limited time offers
  * Event countdowns
  * Launch announcements
  * Sale deadlines

= Use Cases =

* Announce special offers or promotions
* Display important notifications
* Show cookie notices or privacy policy updates
* Highlight upcoming events
* Share news updates
* Display seasonal messages

== Installation ==

1. Upload the `wp-notice-bar` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Notice Bar' in your WordPress admin menu to configure the plugin

== Frequently Asked Questions ==

= Can I display multiple notice bars? =
Yes, you can create and manage multiple notice bars and control when each one appears.

= Is it mobile responsive? =
Yes, the notice bar is fully responsive and looks great on all devices.

= Can I customize the colors? =
Yes, the plugin includes WordPress's native color picker, allowing you to visually select and preview:
* Background colors
* Text colors
Each color picker shows the color code and allows for precise color selection.

= Can I add buttons or links? =
Yes, you can add custom buttons and links to your notice bar.

= Does it support animations? =
Yes, the plugin includes smooth entrance and exit animations for a professional look.
